# Menu de Algoritmos em C

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais da linguagem C, utilizando um sistema de menu interativo no terminal.

O programa permite ao usuário escolher diferentes funcionalidades relacionadas a exercícios básicos de programação.

## Funcionalidades

O menu possui as seguintes opções:

1. Verificar se um número é par ou ímpar
2. Exibir a tabuada de um número
3. Calcular a média de números utilizando vetor
4. Contar a quantidade de letras de uma palavra

## Conceitos utilizados

Este projeto utiliza diversos conceitos importantes da programação em C:

- Estruturas condicionais (`if` / `else`)
- Estruturas de repetição (`for`)
- Vetores
- Manipulação de strings
- Entrada e saída de dados (`scanf` e `printf`)

## Objetivo

O objetivo deste projeto é reforçar o aprendizado de lógica de programação e estrutura de programas em C.

## Tecnologias utilizadas

- Linguagem C
- Compilador GCC
